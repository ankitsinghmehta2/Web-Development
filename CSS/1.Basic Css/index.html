CSS Notes

CSS (Cascading Style Sheets) is used to style HTML elements.

Types of CSS

  • Inline CSS: Applied directly to an element using the style attribute.
  • Internal CSS: Defined within a <style> tag inside the HTML document.
  • External CSS: Stored in an external file and linked using <link>.

Basic CSS Selectors

  • Universal Selector (*) : Targets all elements (e.g., * { margin: 0; }).
  • Element Selector: Targets HTML elements directly (e.g., h1 { color: blue; }).
  • Class Selector: Targets elements with a specific class (e.g., .className { color: red; }).
  • ID Selector: Targets a specific element with an ID (e.g., #idName { color: green; }).

Text Properties

  • Text Alignment: Controls the horizontal alignment of text (e.g., text-align: center;).
  • Text Transformation: Changes text case (e.g., text-transform: uppercase;).
  • Text Decoration: Adds decoration like underline, overline (e.g., text-decoration: underline;).
  • Line Height: Defines space between lines (e.g., line-height: 1.5;).

Font Properties

  • Font Family: Defines the font type (e.g., font-family: Arial, sans-serif;).
  • Font Size: Defines text size (e.g., font-size: 16px;).
  • Font Weight: Sets text thickness (e.g., font-weight: bold;).
  • Font Style: Defines if text is italic, normal, etc. (e.g., font-style: italic;).
